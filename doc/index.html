pimatic sispmctl plugin

Backend for the SIS-PM Control for Linux aka sispmct application that can control GEMBIRD (m)SiS-PM device, witch are USB controled multiple socket.

Configuration

You can load the plugin by editing your config.json to include:

{ 
   "plugin": "sispmctl"
}

in the plugins section. For all configuration options see sispmctl-config-shema

Actuators can be added bei adding them to the devices section in the config file. Set the class attribute to SispmctlSwitch. For example:

{ 
  "id": "light",
  "class": "SispmctlSwitch", 
  "name": "Lamp",
  "outletUnit": 1 
}

For actuator configuration options see the actuator-config-shema file.
